# Image Assets

Required images for the application:

1. **logo.png** - Application logo (200x60px, PNG with transparency)
2. **avatar-placeholder.png** - Default user avatar (100x100px)
3. **dashboard-bg.jpg** - Dashboard background (1920x1080px)
4. **login-bg.jpg** - Login background (1920x1080px)
5. **og-image.png** - Open Graph image (1200x630px)
6. **twitter-card.png** - Twitter card image (1200x600px)

Place your images in this directory.
